②在主页的tabbar中存在三个栏目：
1）时之钟：本页包含了时钟功能，及文字展示。
2）写什么：这个页面包括两个page：“日记”和“Todo”（备忘录)，在这个页面可以进行对日记和todo条目的增删查改。
